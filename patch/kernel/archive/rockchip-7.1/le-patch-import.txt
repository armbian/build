This is a memento in case the LibreELEC repository for rockchip and rockchip-old
patches get merged. At the moment, only the rockchip-old patches are intended
for rk3288 and rk322x, so this procedure is not performed currenlty, but I
keep it here for possible future reference. Jock.

Importing patches from LibreELEC is easy, but may require some manual tidy up.
Since release 12.0, rockchip patches are stored in:

LibreELEC.tv/packages/linux/patches/rockchip

at the moment, there are almost two hundred patches coming mostly from kernel mailing lists.
Since this directory only applies to armhf rockchip devices, we can remove
many patches that only apply to newer chips or to arm64 chips.

After copying the patches into patches.libreelec directory, then these patterns can be
removed:

1. rm *arm64* # mostly removes arm64 device tree adjustments
2. rm *vop2* # removes all the changes to VOP2, which is applicable to rk35xx and above
3. rm *samsung* # no traces of samsung phy in armhf rockchip devices
4. rm *VDPU* # VDPU patches are only for arm64 newer chips
